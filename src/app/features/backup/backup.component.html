Backup & Restore

Export backup Download a full backup of your data. Choose JSON (portable) or SQL (PostgreSQL).
Checking SQL availability... SQL backup requires PostgreSQL and pg_dump on the server.
Restore from backup Upload a backup file to restore data. This replaces all existing data.
Backup format JSON SQL Must match the format of the file you upload

Restore overwrites all company, users, products, orders, and other data. Make a backup first if needed.
